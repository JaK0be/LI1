mapa :: Int -> Int -> [String]

A função mapa gera a Arena de Batalha e as Coordenadas das Bombas e dos Power-Ups (mapa x y retorna uma Arena de Batalha de dimensão x e semente y e as Coordenadas das Bombas e dos Power-Ups).

Exemplos de Utilização:

>>> mapa 11 0
###########
#     ??  #
# # # #?# #
#  ? ?  ? #
#?# # #?# #
# ????   ?#
# #?#?#?#?#
#   ?  ?  #
# # # # # #
#         #
###########
+ 7 1
+ 7 2
+ 7 6
+ 9 6
! 7 4

>>> mapa 9 5
#########
#   ?   #
# #?#?# #
#  ??  ?#
# #?#?# #
#   ?? ?#
# #?# # #
#       #
#########
+ 5 4
! 4 5

mkRandom :: Int -> Int -> [Int]

A Função mkRandom gera aleatoriamente (x*x) números a partir de um número x (Inteiro) e de uma semente y (Inteiro).

chkPwUp :: Int -> String -> [String]

A Função chkPwUp gera as Coordenadas dos Power-Ups (chkPwUp recebe um Inteiro x (dimensão do mapa) e uma string a:as (mapa em que nas posições em que existam Power-Ups, os ' ? ' são substituidos pelo respetivo símbolo de Power-Up).

Exemplo de Utilização:

>>> chkPwUp (quando mapa 11 0)
+ 7 1
+ 7 2
+ 7 6
+ 9 6
! 7 4

>>> chkPwUp (quando mapa 9 0)
+ 5 4
! 4 5

sortStr :: [String] -> [String]

A Função sortStr ordena as Coordenadas dos Power-Ups (sortStr que recebe uma [String] proveniente da função chkPwUp e ordena as Coordenadas dos Power-Ups dessa mesma lista).

sortB :: [String] -> [String]

A Função sortB dá as Coordenadas das Bombas (sortB recebe uma [String] e produz uma outra [String] apenas com as Coordenadas das Bombas).

Exemplos de Utilização:

>>> sortB (quando mapa 11 0)
+ 7 1
+ 7 2
+ 7 6
+ 9 6

>>> sortB (quando mapa 9 5)
+ 5 4

sortF :: [String] -> [String]

A Função sortF dá as Coordenadas das Flames (/sortF/ recebe uma [String] e produz uma outra [String] apenas com as Coordenadas das Flames).

Exemplos de Utilização:

>>> sortF (quando mapa 11 0)
! 7 4

>>> sortF (quando mapa 9 5)
! 4 5

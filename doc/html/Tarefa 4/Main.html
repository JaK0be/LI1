A Função avanca faz decorrer um 1 instante de tempo no jogo

checktime :: [String] -> Int -> [String]

A Função checktime tem como objetivo verificar de o tempo de jogo é inferior a (n-2)^2(Sendo n a dimensão do mapa). Caso seja igual vai dar início à espiral, caso contrário remove 1 unidade de tempo.

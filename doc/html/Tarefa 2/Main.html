comando :: [String] -> Char -> String -> String

A Função comando é uma função que verifica qual o comando que se pretenda que seja executado e verifica se é possível executar esse comando (Verifica se é possível mover-se para uma "dada posição", e caso seja verifica se existe algum tipo de powerUp e consequentemente move o respetivo Jogador).

chkmove :: [String] -> String -> Char -> Bool

A Função chkmove verifica se o movimento para uma determinada posição é válido (Se a posição para o qual pretendemos mover for '#' ou ' ? ', esta função retribui um False impedindo assim a ação).

posGet :: String -> (Int, Int)

A Função posGet recebe uma String e produz um (Int,Int)(O (Int,Int) fornecido por esta função corresponde às Coordenadas para as quais um Jogador se pretende mover).
